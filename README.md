# Defense Shield Simulator

A multi-layer authentication simulator with an interactive game-based lockout system.

## Features
- Username & password verification
- Progressive security layers (2FA + security question)
- Account lock after 3 failed attempts
- Tic Tac Toe game system:
  - Easy mode (random bot)
  - Hard mode (smart bot)

## Tech Stack
- HTML
- CSS
- JavaScript
- C++ (logic prototype)
